### node-finance

An equity-based financial framework for Node.js.

Plan to include support for the following:

* Stocks
* Bonds
* Options
* Forex
* Futures

### Install
	git clone git@github.com:ChromoX/node-finance.git
	cd node-finance
	npm install -d
	node app
