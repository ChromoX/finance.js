### node-finance

**node-finance** is an equity-based financial framework for Node.js.

Plan to include tools, utilities, and functions for the following:
* Stocks
* Bonds
* Options
* Forex

### Install
	git clone git@github.com:ChromoX/node-finance.git
	cd node-finance
	npm install -d
	node app
